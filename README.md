# Student Mark Checker

## Description

Student Mark Checker is a simple web application that allows users to enter a student's name and mark, calculate the result, determine pass/fail status, and display a list of submitted students.

## Features

* Enter student name and mark
* Validate user input
* Calculate grades automatically
* Display pass or fail status
* Store submitted students in a list
* Responsive and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
